# Terminal control in code with hashing

Console user manager on Node.js using PostgreSQL and secure password hashing via Bcrypt.

## Quick start
1. **Install dependencies:** `npm install`
2. **Configure `.env`:** Add the line `DB_URL=your_database_url`
3. **Run the command:** `node room.js [command]`

## Available commands
- `list` — list of users.
- `register` — registration.
- `login` — login.
- `update` — change password.
- `delete` — delete account.
